# Job Application Portal Test Framework

## Overview
This repository contains the automation framework for testing the Job Application Portal. Built with Selenium, API testing, and CI/CD for comprehensive SDET coverage.

## Features
- Manual test plans and execution
- Selenium-based UI automation
- API testing with pytest
- CI/CD integration with GitHub Actions
- Comprehensive test case management

## Setup
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Set up the Job Application Portal (see main repo)

## Usage
- Run manual tests: Follow test plans in `test-plan/`
- Execute automated tests: `pytest tests/`
- View CI results: GitHub Actions

## Project Structure
- `docs/`: Documentation and plans
- `test-plan/`: Manual test cases
- `tests/`: Automated test scripts

## Contributing
1. Follow the roadmap in docs/
2. Add test cases to appropriate folders
3. Ensure CI passes before merging
